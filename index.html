RTCRtpReceiver extensions 

partial interface RTCRtpReceiver {
      attribute RTCLatencyCategory latencyHint;
};

Attributes

latencyHint of type RTCLatencyCategory, initialized to "interactive" .

This attribute affects tradeoffs between media quality and latency. It is up to the browser how best to handle this preference.

RTCLatencyCategory Enum

enum RTCLatencyCategory {
        "balanced",
        "interactive",
        "playback"
};
Enumeration description
balanced Balance media quality and latency.
interactive Provide the lowest latency possible without noticeable media distortions such as audible speech expansions or video freezes. This is the default.
playback Prioritize quality by increasing latency.
